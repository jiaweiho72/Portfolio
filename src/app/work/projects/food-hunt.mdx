## Overview

Developed a web application aimed at reducing food wastage by connecting vendors with consumers. The vendor interface allows users to upload unsold food items, while the consumer interface enables users to search for and reserve available listings.

## Technologies Used
- Frontend: Vuejs
- Backend: Firebase


## Key Features

- **Vendor Dashboard**: A user-friendly interface for vendors to list unsold food items quickly and efficiently.
- **Search Functionality**: Consumers can easily search for available food listings based on location, category, and availability.
- **Reservation System**: Users can reserve food items, ensuring better coordination between vendors and consumers and minimizing waste.

## Challenges Faced

Encountered challenges in implementing real-time data updates and ensuring accurate inventory management. Overcame these issues by leveraging Firebase's real-time database capabilities.

## Outcome

This project enhanced my skills in web application development and reinforced the importance of technology in addressing social issues such as food waste. It demonstrates my ability to create practical, user-centered solutions that promote sustainability.
